# Markdown 사용법
## 개행
문장과 문장 사이에 스페이스바 2회  
문단과 문단 사이의 빈 줄  

```
그냥 엔터 키만 입력하면
줄이 띄워지지 않는다.

스페이스바 2번 입력하면  줄이 띄워진다.
```

## 코드 블록(Code Blocks)
```
'''
따옴표 안에 코드를 입력.  
언어를 명시하면 색상이 자동으로 조절된다.
'''
''' java
System.out.println("Hello World!");
'''
    줄의 맨 앞에 스페이스 4개
```
```
따옴표 안에 코드를 입력.

```
```java

```
